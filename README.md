# ai_ml_project
Arindam Mohanty 25BCE11311 AI ML Project
A Python utility for evaluating password integrity and generating complex credentials.

Features -

Strength Metric: Classifies passwords as Weak, Medium, or Strong.

Generator: Builds random, high-security strings.

Vault: Logs generated passwords to a local file.

Setup -

Download the script files.

Execute python main.py in your terminal.
